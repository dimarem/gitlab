# Gitlab

Различные примеры работы с Gitlab и Yandex Cloud.

Для развертывания Gitlab используется код из этого [репозитория](https://github.com/dimarem/terraform) (gitlab), с помощью которого создаются следующие ресурсы:

- экземпляр Gitlab
- сервисный аккаунт, от имени которого будут выполняться действия в облаке
- авторизационный ключ доступа сервисного аккаунта
- сеть и подсеть, в которой будет размещаться экземпляр Gitlab
- виртуальная машина, на которой будет запущен раннер
- сервис для хранения и распространения Docker-образов (YCR)
- сервис для хранения секретов (Lockbox)
- ключ Yandex Key Management Service для шифрования секретов в Lockbox

> ⚠️ Все ссылки на ресурсы в примерах этого репозитория относятся к созданным ресурсам выше.
